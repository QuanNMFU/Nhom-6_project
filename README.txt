How to run the Online Food Ordering System Using PHP and MySQL

Software:	XAMPP / Wamp / Mamp / Lamp (anyone)

1.Download the project folder (OnlineFood and SQL File)
2.Copy OnlineFood folder then paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
3.Open PHPMyAdmin (http://localhost/phpmyadmin)
4.Create a database with name 'onlinefoodphp'
5.Import onlinefoodphp.sql file (given inside the zip package in SQL File folder)
6.Run the script http://localhost/OnlineFood
